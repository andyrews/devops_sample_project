# devops_sample_project
## DevOps Practice with:
- Postgre
- Redis Cache
- API Gateway
- Github Actions (Jenkins Later)
- Clerk Auth

## Architecture
 ![Architecture](assets/Architecture.drawio.png) 

## Initial Usage
### Frontend
```sh
npm i
npm run dev
```

### Backend
```sh
make install (per microservice)
python server.py (per microservice)
```

### Redis
```sh
redis-server --port 6379
```
